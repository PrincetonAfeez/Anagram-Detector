# Schema

Simple JSON Schema files for the Anagram Detector project.

## Files

- `input.schema.json` — validates the two words or phrases being compared.
- `result.schema.json` — validates the anagram detection output.
- `config.schema.json` — validates optional detector settings.
- `test-case.schema.json` — validates reusable test cases.
- `index.schema.json` — collection-level schema that references the other schema files.

These schemas use JSON Schema Draft 2020-12.
